# Neural-Network
This project shows the fundamental of Neural Networks by implementing a multi-layer perceptron entirely from scratch without the use of standard deep learning libraries. The custom implementation is then benchmarked against an optimized standard library model (Keras/TensorFlow) across two distinct classification datasets: the relatively simple IRIS dataset (Iris flowers species) and the more complex MNIST dataset (handwritten digits images).

The goal is to provide a deep understanding of network architecture, forward and backward propagation, multi-class vs. binary classification nuances, and the practical advantages of modern machine learning frameworks.
